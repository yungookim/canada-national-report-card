# Canada's 2025 National Dashboard

A highly visual, animated website presenting the 2025 National Report.

## How to Run

1.  Open Terminal.
2.  Navigate to the project folder:
    ```bash
    cd /Users/dgyk/.gemini/antigravity/scratch/canada-2025-dashboard
    ```
3.  Install dependencies (if you haven't already):
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```
5.  Open your browser to the URL shown (usually `http://localhost:5173`).

## Features

-   **Responsive Design**: Works on mobile and desktop.
-   **Scroll Animations**: Sections fade in as you scroll.
-   **Data visualization**: Key stats (Inflation, Employment, Housing) highlighted with animated counters and clear "Good/Bad/Neutral" indicators.
-   **Accessibility**: Written for Grade 9 reading level with simple terms and emojis.
